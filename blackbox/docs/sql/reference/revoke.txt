==========
``REVOKE``
==========

Revokes privilege from an existing user.

Synopsis
========

.. code-block:: psql

  REVOKE { { READ | WRITE [,...] } | ALL [ PRIVILEGES ] }
  ON { [ TABLE ] table_name [, ...] | ALL TABLES IN SCHEMA schema_name [, ...] }
  FROM user_name [, ...];


Description
===========

``REVOKE`` is a management statement to revoke a ``READ`` and/or ``WRITE`` privilege
on one or many tables, or all tables in a schema from one or many existing existing
users.

For usage of the ``GRANT`` statement see
:doc:`/sql/administration/privileges`.

Parameters
==========

:table_name: The name of an existing table.

:schema_name: The name of an existing schema.

:user_name: The name of an existing user.
